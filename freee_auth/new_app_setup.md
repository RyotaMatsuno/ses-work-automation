# freee新規アプリ作成手順

## アクセスURL
https://app.secure.freee.co.jp/developers/applications/new

## 入力値

| 項目 | 値 |
|---|---|
| アプリ名 | TERRA SES管理 |
| アプリの説明 | SES事業の請求書・入金管理自動化用 |
| コールバックURL | urn:ietf:wg:oauth:2.0:oob |
| スコープ | 会計: 読み取り・書き込み（請求書・取引）|

## 作成後にジョブズへ貼る情報
- Client ID
- Client Secret

## 注意
- 「下書き保存」ではなく「保存」または「作成」をクリック
- 「申請して公開」は不要（プライベートアプリのまま使用）
