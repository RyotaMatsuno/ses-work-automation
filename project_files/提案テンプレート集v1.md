## テンプレート5: サマリー用アピール文ルール

| 条件 | サマリーアピール文 |
|---|---|
| 必須全○ + 尚可全○ | 必須・尚可ともにマッチ度高い人員 |
| 必須全○ + 尚可○率50%以上 | 必須全て満たしており、尚可も{○項目}経験あり |
| 必須全○ + リーダー経験あり | 必須全て満たしており、リーダー経験あり |
| 必須全○ + 上流経験あり | 必須全て満たしており、上流工程経験あり |
| 必須全○ + 単価優位な若手 | 必須全て満たしており、単価面で優位な若手 |
| 必須全○のみ | 必須スキル全て満たし即稼働可能 |

### 注意事項
- 「充足」は使わない →「全て満たしており」
- 「即戦力です」は使わない →「マッチ度高い人員かと存じます」
- 「本命」「チャレンジ枠」等のラベルはメールに記載しない

---

## テンプレート6: 並行状況の記載ルール

- 面談調整中: 「1件（面談調整中）」
- 面談予定: 「1件（面談予定）」
- 結果待ち: 「1件（結果待ち）」
- なし: 「なし」
- 「提案中」は記載しない（業界では並行とみなさない）

---

## テンプレート7: 面談パターン別の人材への説明文

- パターンA（1回面談）: 「面談1回で完結する案件です。」
- パターンB（2回・人柄型）: 「2回面談あり、1回目は人柄確認、2回目が本番です。」
- パターンC（2回・エンド顔合わせ型）: 「1回目が実質の本番、2回目はエンドユーザーへの顔合わせです。」

---

## テンプレート8: 敬語パターン

- 通常: 「何卒よろしくお願いいたします。」
- 急ぎ: 「お忙しいところ恐れ入りますが、ご確認のほどよろしくお願いいたします。」
- 確認依頼: 「ご検討いただけますと幸いです。何卒よろしくお願いいたします。」
