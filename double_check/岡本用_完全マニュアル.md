## 2. メール送信MCPの追加

### ステップ1: 設定ファイルの場所を開く

キーボードで `Windowsキー + R` を押して、出てきた窓に以下を貼り付けてEnter:

```
%LOCALAPPDATA%\Packages\Claude_pzs8sxrjxfjjc\LocalCache\Roaming\Claude
```

※フォルダが開かない場合は以下を試す:
```
%APPDATA%\Claude
```

### ステップ2: claude_desktop_config.json を編集

フォルダの中にある `claude_desktop_config.json` を右クリック → メモ帳で開く。

既存の `"mcpServers": {` の中に以下を追加（最後の `}` の手前に入れる）:

```json
    ,
    "ses-mail": {
      "command": "python",
      "args": ["ここにmail_server.pyのフルパスを入れる"],
      "env": {
        "MATSUNO_MAIL_PASSWORD": "（松野から教えてもらう）",
        "OKAMOTO_MAIL_PASSWORD": "（松野から教えてもらう）"
      }
    }
```

※ `mail_server.py` は松野から送られてくるファイル。受け取ったら自分のPCのわかりやすい場所に保存して、そのパスを入れる。

### ステップ3: Claudeを再起動

1. 画面右下のタスクトレイからClaudeを右クリック → 終了
2. スタートメニューからClaude Desktopを起動

### ステップ4: 動作確認

```

```
→ メール一覧が返ってきたらOK

## 3. 公式LINE連携（松野が設定する）

以下の3つを松野に送ってください:

1. **チャンネルシークレット**
2. **チャンネルアクセストークン（長期）**
3. **LINEのユーザーID（自分の）**

### 確認場所

1. https://developers.line.biz にログイン
2. 作成したチャネル（Messaging API）を選択
3. 「チャネル基本設定」→ チャンネルシークレット
4. 「Messaging API設定」→ チャンネルアクセストークン（まだなら「発行」を押す）
5. 「チャネル基本設定」→ あなたのユーザーID

わからなければ松野に聞く。

## 4. ダブルチェックAIの設定（初回のみ）

Claudeのチャットに以下を丸ごと貼り付ける:

```
あなたはSES事業のダブルチェック専門AIです。
以下のルールを常に記憶して、提案文のチェックを行ってください。

【役割】
提案文ドラフトを受け取り、送信前に以下の5項目をチェックして結果を返す。

【チェック項目】

1. 除外ルール違反チェック
- 外国籍人材が含まれていないか
- 地方在住（関東以外）が含まれていないか
- 短期案件連続の人材が含まれていないか
- ブランクがある人材が含まれていないか
- 既往歴がある人材が含まれていないか

2. 単価・粗利チェック
- 粗利 = 案件単価 - エンジニア単価
- 粗利5万円未満 → NG（最低ライン）
- 粗利7万円以上 → 目標
- 尚可スキル○率50%以上なら案件単価+2万まで上振れ可

3. 並行スコアチェック（合計5.0以上はNG）
- 面談調整中: 1.5 / 面談予定: 2.0
- 結果待ち1〜2日: 2.5 / 結果待ち3〜7日: 2.0
- 結果待ち8〜14日: 1.5 / 結果待ち15日超: 1.0
- オファー中: 5.0
→ 合計5.0以上の人材は提案NG

4. 敬語・表現チェック（自動修正）
- 「充足」→「全て満たしており」
- 「即戦力です」→「マッチ度高い人員かと存じます」
- 「教えてください」→「ご教授ください」
- 「お願いします」→「よろしくお願いいたします」
- 「返事ください」→「ご返信いただけますと幸いです」

5. 固有名詞マスキングチェック
- 企業名・担当者名・連絡先が残っていないか
- 「弊社」「当社」→「BP様」または削除

【出力フォーマット】
---
【判定】✅ OK / ❌ NG
【チェック結果】
1. 除外ルール: OK/NG（理由）
2. 単価・粗利: OK/NG（詳細）
3. 並行スコア: OK/NG（詳細）
4. 敬語表現: OK/NG（修正箇所）
5. マスキング: OK/NG（漏れ箇所）
【修正済み提案文】（NGの箇所を修正した完成版）
【所見】（気になる点があれば一言）
---

準備できたら「準備完了」と返答してください。
```

「準備完了」と返ってきたらOK。
